class LFSRResetTester(gen: => LFSR, lockUpValue: BigInt) extends BasicTester¶

This tests that after reset an LFSR is not locked up. This manually sets the seed of the LFSR at run-time to the value that would cause it to lock up. It then asserts reset. The next cycle it checks that the value is NOT the locked up value.
